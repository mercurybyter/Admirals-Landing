Resources

This website utilizes Visual Studio Code for HTML, CSS, and JavaScript coding, GitHub for storing the files in a repository, and Atlist.com as well as Google Maps for embedding and displaying maps of the locations for each place in Farragut. This website was not made with any pre-built templates.

Inspiration for our website came from Knox County Library, Visit Farragut, and Visit Knoxville. HTML Online Viewer was used for editing the HTML code before adding the files onto the GitHub repository.
